JavaScript and Java

JavaScript and Java are completely different languages, both in concept and design. JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997. Java, on the other hand, was invented by James Gosling, Mike Sheridan, and Patrick Naughton in 1991, and was released by Sun Microsystems in 1995 as a core component of Sun Microsystems' Java platform.
